## 🏁 Getting Started

1. 📥 Clone this repository

**SSH**

```
git@github.com:Kmqe/reda-online-store.git
```

**HTTPS**

```
https://github.com/Kmqe/reda-online-store.git
```

2. Install dependencies

```
npm install
```

3. Start the development server:

```
npm run dev
```

4. 🎉 Enjoy!
